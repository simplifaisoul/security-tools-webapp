# Understanding Hearts - Website

A compassionate guide to understanding and supporting upset ladies with empathy, respect, and care.

## Files
- `upset-ladies-guide.html` - Main HTML file
- `upset-ladies-styles.css` - CSS styling
- `upset-ladies-script.js` - JavaScript functionality

## Features
- Responsive design for all devices
- Smooth scrolling navigation
- Interactive animations
- Mobile-friendly hamburger menu
- Modern gradient design

## Publishing Instructions

### GitHub Pages
1. Create a new GitHub repository
2. Upload all three files
3. Go to Settings > Pages
4. Select main branch and save
5. Your site will be live at `https://username.github.io/repository-name`

### Netlify
1. Go to netlify.com
2. Drag and drop the files folder
3. Your site will be instantly published

### Vercel
1. Go to vercel.com
2. Import the files
3. Deploy with one click

## Local Development
Run `python3 -m http.server 8000` and visit `http://localhost:8000/upset-ladies-guide.html`
